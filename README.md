# Python Projects 🐍

A collection of simple, beginner-friendly Python mini-projects built for practice and learning purposes.

## Projects Included

### 1. Calculator
A simple command-line calculator that performs basic arithmetic operations like addition, subtraction, multiplication, and division.

### 2. Password Generator
A Python script that generates strong, random passwords based on user-defined length and character preferences.

### 3. Tic Tac Toe
A classic two-player Tic Tac Toe game playable in the terminal.

### 4. To-Do List
A simple to-do list application to add, view, and manage daily tasks from the command line.

## How to Run

1. Clone the repository
```bash
   git clone https://github.com/nikhil-kumarrr/Python-Projects.git
```
2. Navigate into the desired project folder
```bash
   cd Python-Projects/"Project Name"
```
3. Run the Python file
```bash
   python filename.py
```

## Requirements

- Python 3.x installed on your system

This repository is a beginner-friendly practice space to explore and strengthen core Python concepts through small, hands-on projects.
